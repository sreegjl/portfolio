The 4 kinematic equations

All four assume constant acceleration. There are five variables total — \(x\), \(v_0\), \(v\), \(a\), and \(t\) — and each equation is missing one of them. When solving a problem, identify which variable you don't know and don't need, then use the equation that's missing that variable.

Equations — pick by missing variable

Missing \(x\) \(v = v_0 + at\)
Missing \(v\) \(x = x_0 + v_0 t + \tfrac{1}{2}at^2\)
Missing \(t\) \(v^2 = v_0^2 + 2a(x - x_0)\)
Missing \(a\) \(x - x_0 = \tfrac{1}{2}(v_0 + v)t\)

The 5 variables

\(x\) displacement
\(v_0\) initial velocity
\(v\) final velocity
\(a\) acceleration
\(t\) time

Free fall

An object is in free fall when gravity is the only force acting on it — air resistance is negligible. All free-falling objects near Earth's surface accelerate at the same constant rate regardless of mass, denoted \(g = 9.8\,\text{m/s}^2\) downward. The kinematic equations apply directly, with \(a = -g\) if up is positive.

Key facts

Acceleration due to gravity \(g = 9.8\,\text{m/s}^2\) downward
Dropped from rest \(v_0 = 0\)
At the peak \(v = 0\), but \(a = -g\) still
Acceleration on the way up still \(-g\) downward

Thrown upward — signs (up = +)

On the way up \(v > 0\), \(a < 0\) — slowing
At the top \(v = 0\), \(a < 0\)
On the way down \(v < 0\), \(a < 0\) — speeding up
\(v\) vs. \(t\) graph straight line, slope \(= -g\)
\(y\) vs. \(t\) graph concave-down parabola

Vectors in 2D

A 2D vector can be broken into two perpendicular components along the \(x\) and \(y\) axes. The total vector and its components form a right triangle — so trigonometry and the Pythagorean theorem apply directly.

Projectile motion

A projectile moves in two dimensions with gravity as the only force — free fall plus horizontal motion. The key insight is that the horizontal and vertical dimensions are independent: analyze them separately using the kinematic equations, with the same \(t\) in both.

Time in the air is determined entirely by vertical motion. Horizontal speed affects range, not hang time.

Horizontal (ax = 0)

Velocity \(v_x = v_{x0}\) (constant)
Position \(x = x_0 + v_{x0}t\)
Acceleration \(a_x = 0\)

Vertical (ay = −g)

Velocity \(v_y = v_{y0} - gt\)
Position \(y = y_0 + v_{y0}t - \tfrac{1}{2}gt^2\)
No-time \(v_y^2 = v_{y0}^2 - 2g\Delta y\)
Launched horizontally \(v_{y0} = 0\)

Key facts

At the peak \(v_y = 0\), \(v_x\) unchanged
Total speed at peak \(v = v_x\) only
Hang time depends on \(v_{y0}\), \(\Delta y\), \(g\) only
\(t\) is a scalar same value in both \(x\) and \(y\)
Trajectory shape parabola

Symmetry (lands at same height)

Time up = time down \(\Delta t_\text{up} = \Delta t_\text{down}\)
\(v_y\) at same height equal magnitude, opposite sign
Net vertical displacement \(\Delta y = 0\)

Quick-reference identities

Total speed \(v = \sqrt{v_x^2 + v_y^2}\)
Horizontal component \(v_x = v\cos\theta\)
Vertical component \(v_y = v\sin\theta\)
Speed at peak \(v_\text{top} = v_x = v\cos\theta\)

When not to use kinematics

\(a = 0\) (horizontal motion) just use \(d = vt\)
\(a \neq 0\) (vertical motion) use kinematic equations
Finding \(t\) always solve from \(y\) first

Problem-solving strategy

1 Split all vector quantities into \(x\) and \(y\) components
2 Find time \(t\) using whichever direction gives enough info — usually \(y\) since that's where \(a \neq 0\)
3 Use that same \(t\) in the other direction
4 At the peak: \(v_y = 0\), so \(v = v_x\)
5 If launch height = landing height: \(t_\text{up} = t_\text{down}\) and \(v_y\) has equal magnitude but opposite sign at matching heights
